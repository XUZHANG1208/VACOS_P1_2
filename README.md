# VACOS_P1_2
In this step, Bayesian modeling will be embedded into the pipeline and more tests are going on

The VACOS line2 will only include brms logistic regression models, both binomial and multinomial.
The line 2 repository includes "get_posterior_samples", "cal_brms_stas" and the "VACOS_line2_Bayes" workflow. 
This workdlow can generate coefficient estimates of all fixed effects in brms logistic regression models, brms model evaluation scores and posterior samples.
